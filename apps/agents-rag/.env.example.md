```bash
# 本地运行（仅本地用 .env；线上 Workers 不会自动读取 .env）
VOLCENGINE_API_KEY=your_volcengine_api_key
VOLCENGINE_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
VOLCENGINE_CHAT_MODEL=ep-xxxxxxxxxxxxxxxxxxxx
VOLCENGINE_EMBEDDING_MODEL=ep-xxxxxxxxxxxxxxxxxxxx
RAG_VECTOR_STORE_KEY=vector_store.json
```

## 部署到 Cloudflare Workers

- 非敏感配置放在 wrangler.toml 的 [vars]（本项目已配置 baseUrl/chatModel/embeddingModel/vectorStoreKey）
- API Key 用 secret 配置（不要写进 wrangler.toml）

```bash
cd apps/agents-rag
wrangler secret put VOLCENGINE_API_KEY
wrangler deploy
```

部署后可用 `GET /rag/health` 检查当前生效的 `chatModel/embeddingModel` 是否为 `ep-xxxx`。
